# Smart Expense Tracker

A modern and interactive Expense Tracker web application built using HTML, CSS, and JavaScript.

## Features

- Add Income and Expenses
- Expense Categories
- Interactive Dashboard
- Analytics Pie Chart
- Progress Tracking
- Responsive Design
- Modern UI with Animations

## Technologies Used

- HTML
- CSS
- JavaScript
- Chart.js

## How to Run

1. Download the project
2. Open index.html in browser

- Login System
